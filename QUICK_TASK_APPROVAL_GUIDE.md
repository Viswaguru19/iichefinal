# Quick Task Approval Guide

## For EC Members: How to Approve Tasks

### Method 1: From Event Progress Page
1. Go to **Dashboard → Event Progress**
2. Select an event from the list
3. Look for yellow button: **"X Tasks Awaiting Approval"** (if tasks pending)
4. Click the button to go to Event Detail page
5. Review and approve tasks

### Method 2: Direct to Event Detail
1. Go to **Dashboard → Event Progress**
2. Click on any event
3. Click **"View Details"** or navigate to event detail page
4. Scroll to **"Tasks Awaiting EC Approval"** section (yellow highlight)
5. Review and approve tasks

## Approval Options

### Option A: Approve As-Is
- Click **"Approve Task"** button
- Task is immediately assigned to committee
- Committee receives notification

### Option B: Edit & Approve
- Click **"Edit & Approve"** button
- Modify task title and/or description
- Click **"Save & Approve"**
- Modified task is assigned to committee
- Committee receives notification

## What Happens After Approval?

1. ✅ Task status changes to `approved`
2. 📊 Task appears in Event Progress dashboard
3. 🔔 All members of assigned committee get notified
4. 📋 Task shows in committee's Tasks section
5. ✨ Committee can start working on the task

## Visual Indicators

| Indicator | Meaning |
|-----------|---------|
| 🟨 Yellow highlight | Task pending EC approval |
| ✅ Green badge | Task approved by EC |
| 🔵 Blue badge | Task in progress |
| ✔️ Green checkmark | Task completed |
| ⚠️ Pulsing button | Action required |

## Quick Tips

- **Pending tasks don't count** toward event progress
- **Only approved tasks** are visible to committees
- **You can edit** task details before approving
- **Notifications** are sent automatically
- **Progress tracking** starts after approval

## Common Questions

**Q: Can I reject a task?**
A: Currently, you can only approve. Rejection feature coming soon.

**Q: Can I reassign a task to a different committee?**
A: Yes, use "Edit & Approve" to modify the assignment.

**Q: What if I approve by mistake?**
A: Contact admin to revert. Future update will add undo feature.

**Q: Do I need to approve every task?**
A: Yes, all tasks require EC approval before assignment.

**Q: Can multiple EC members approve the same task?**
A: Only one EC approval is needed. First approval wins.
